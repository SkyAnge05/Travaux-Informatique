# Travaux Informatique

**Auteur :** Nelen Nathan  
**Année d'étude :** 2024 - 2025

Bienvenue dans le dépôt de mes travaux d'école ! C'est ici que je stocke et organise tout le code que je réalise dans le cadre de mes cours d'informatique, principalement axés sur le langage C.

## 📂 Structure du projet

Afin de faciliter la navigation, les fichiers sont organisés par notions et types de travaux :

*   **`Docs/`** : Documents de référence, aide-mémoires pour les examens et énoncés.
*   **`Labo_Fonction/`** : Projets et laboratoires centrés sur l'utilisation des fonctions.
*   **`Labo_Tableau-Pointeur/`** : Laboratoires combinant la manipulation des tableaux et des pointeurs.
*   **`Les_Structures/`** : Exercices et travaux pratiques sur les structures de données en C.
*   **`Multi fichier/`** : Projets démontrant la programmation modulaire (séparation en fichiers `.c` et `.h`).
*   **`Pointeur/`** : Exercices dédiés à la compréhension et l'usage des pointeurs.
*   **`Tableau/`** : Exercices sur les tableaux et la manipulation de chaînes de caractères.
*   **`TPs/`** : Fichiers sources des travaux pratiques généraux.
*   **`Teste/`** : Dossier de brouillon contenant des tests, des essais de code et des préparations.
*   **`output/`** : Dossier regroupant la plupart des fichiers exécutables compilés.

## 🛠️ Outils utilisés

*   Langage de programmation : **C**
*   Environnement de travail configuré pour Visual Studio Code.
